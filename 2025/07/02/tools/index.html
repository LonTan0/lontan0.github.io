2.hydra

hydra -L up.txt -p up.txt ssh://192.168.56.104:2222 -V

参数 作用
-L 指定用户名字典
-l 指定用户名
-P 指定密码字典

-V 输出每一次尝试爆破的用户名和密码以及结果

4.dig

dig（Domain Information Groper）是一个DNS 查询工具

类型 含义
A 主机地址（IPv4）
AAAA 主机地址（IPv6）
MX 邮件服务器记录
NS 授权名称服务器
CNAME 规范名称（别名）
TXT 文本记录（SPF/验证信息）
SOA Start of Authority（起始权威）
AXFR 区域传送（Zone Transfer）
参数 功能说明
+short 只显示简洁的答案（比如 IP 地址）
+tcp 使用 TCP 协议（⚠️AXFR 必须使用 TCP）
+noquestion 不显示查询问题部分（请求行）
+nocomments 不显示注释
+nostats 不显示统计信息
+nocmd 不显示命令提示头
+multiline 美化输出格式，多行展示
+trace 启用跟踪模式，从根服务器逐级查询域名

5.searchsploit

searchsploit 是一个命令行工具，用于在本地快速搜索 Exploit Database 中已知的漏洞利用代码（PoC）和漏洞信息。

参数 说明
-u 更新本地数据库（相当于 git pull）
-p 显示 Exploit 的完整路径
-x <路径> 直接显示 PoC 文件内容
-m <路径> 复制 PoC 到当前目录
--json 输出为 JSON 格式，便于脚本处理
